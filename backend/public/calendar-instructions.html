Instructions pour Google Calendar

Pour connecter votre compte Google Calendar à l'application de téléconsultation, veuillez suivre ces étapes :

  1. Cliquez sur le bouton "Connecter à Google Calendar" ci-dessous.
  2. Sélectionnez le compte Google que vous souhaitez utiliser pour vos rendez-vous de téléconsultation.
  3. Sur l'écran d'autorisation, assurez-vous de cocher toutes les cases pour permettre à l'application d'accéder à votre calendrier.
  4. Cliquez sur "Continuer" ou "Autoriser" pour terminer le processus.
Note importante : Si vous avez précédemment refusé l'accès, vous devrez peut-être d'abord révoquer les autorisations précédentes. Pour ce faire, visitez votre compte Google et supprimez l'application de téléconsultation de la liste des applications connectées.
